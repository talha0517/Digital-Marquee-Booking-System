# Digital Marquee Booking System

## Overview
Digital Marquee Booking System is a Java-based desktop application developed to simplify marquee reservation and event management. The system allows users to view available marquees, make bookings, manage event details, and maintain customer records efficiently.

## Features
- User-friendly graphical interface
- Marquee booking and reservation management
- Customer information management
- Event scheduling
- Booking status tracking
- Database integration for data storage
- Object-Oriented Design

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- Object-Oriented Programming (OOP)

## Project Structure
- User Management
- Booking Management
- Event Scheduling
- Database Connectivity
- GUI Components

## How to Run
1. Clone the repository:
   git clone https://github.com/talha0517/Digital-Marquee-Booking-System.git

2. Open the project in Netbeans.

3. Configure the MySQL database.

4. Update database credentials in the source code.

5. Compile and run the application.
